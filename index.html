Insertion Sort Visualizer

Insertion sort is one of the common sorting algorithms in computer science. Although, it is not that effiecient considering it makes \(O(n^2)\) comparisions in the worst case where \(n\) is the size of the data, it is a better sorting algorithm compared to bubble sort and is fairly easy to implement . To visualize how it works, press the sort button and the sticks will sort themselves. Try different shuffles by pressing shuffle button. However, beware, you cannot shuffle while the array is being sorted.
